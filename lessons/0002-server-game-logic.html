Overview: The Game Loop

The game progresses through a state machine. Here's the lifecycle of a game:

NOT_STARTED (lobby) │ │ startGame() ▼ CHOOSING_WORD ──► nextRound() sends words to drawer │ │ wordSelected() or timeout auto-selects ▼ DRAWING (drawTime seconds) │ │ All guessed? Timer up? Player left? ▼ endRound() → givePoints() → TURN_END event │ │ More rounds? ──► nextRound() │ No ▼ endGame() → WINNER event → auto-restart (public rooms)

The Entry Point: socketHandlers.ts

This file is the router. Every socket event is wired to a handler. There's no business logic here — just validation and delegation:

socket.on(GameEvent.START_GAME, async ({ words }) => {
  const room = await getRoomFromSocket(socket);
  // Validates: is the host? game not started? 2+ players?
  await startGame(room, io);
});

Notice: getRoomFromSocket(socket) gets the game room by reading the socket's second joined room (the first is the default socket room, the second is the game room):

// gameController.ts
export async function getRoomFromSocket(socket: Socket) {
  const roomId = Array.from(socket.rooms)[1] as string; // [1] = game room
  const room = await getRedisRoom(roomId);
  return room;
}

startGame() — The Starting Gun

export async function startGame(room: Room, io: Server) {
  clearTimers(room.roomId);                   // Clear any stale timers
  room.gameState.currentRound = 1;            // Start at round 1
  room.gameState.currentPlayer = 0;           // First player in array
  room.gameState.guessedWords = [];
  room.gameState.drawingData = [];
  room.gameState.hintLetters = [];
  room.gameState.roomState = RoomState.CHOOSING_WORD;
  await setRedisRoom(room.roomId, room);
  io.to(room.roomId).emit(GameEvent.GAME_STARTED, room);
  await nextRound(room.roomId, io);           // Start first round
}

Key detail: The host's custom words (if any) are already saved to room.settings.customWords before this is called, back in the socket handler.

nextRound() — Picking the Drawer

export async function nextRound(roomId: string, io: Server) {
  const room = await getRedisRoom(roomId);
  const currentPlayer = room.players[room.gameState.currentPlayer];

  const words = await getRandomWords(
    room.settings.wordCount,      // default 3
    room.settings.language,
    room.settings.onlyCustomWords,
    room.settings.customWords
  );

  // Send word options ONLY to the drawer
  io.to(currentPlayer.playerId).emit(GameEvent.CHOOSE_WORD, { words, time: WORDCHOOSE_TIME });

  // Tell everyone else "X is choosing a word"
  io.to(room.roomId)
    .except(currentPlayer.playerId)
    .emit(GameEvent.CHOOSING_WORD, { currentPlayer, time: WORDCHOOSE_TIME });

  // Auto-select a random word if drawer doesn't pick in 30 seconds
  const timeOut = setTimeout(async () => {
    if (room.gameState.word != "") return; // already picked
    const randomWord = words[Math.floor(Math.random() * words.length)];
    await wordSelected(roomId, randomWord, io);
  }, WORDCHOOSE_TIME * 1000);
  timers.set(roomId, timeOut);
}

wordSelected() — Starting the Drawing Phase

export async function wordSelected(roomId: string, word: string, io: Server) {
  const room = await getRedisRoom(roomId);
  clearTimers(room.roomId);

  room.gameState.word = word;
  room.gameState.roomState = RoomState.DRAWING;
  room.gameState.timerStartedAt = new Date();
  await setRedisRoom(roomId, room);

  const player = room.players[room.gameState.currentPlayer];

  // Drawer sees the actual word
  io.to(player.playerId).emit(GameEvent.WORD_CHOSEN, { word, time: room.settings.drawTime });

  // Guessers see letter counts (e.g. "elephant" → [8])
  const words_lens = convertToUnderscores(word);
  io.to(room.roomId).except(player.playerId).emit(GameEvent.GUESS_WORD_CHOSEN, {
    word: words_lens,
    time: room.settings.drawTime,
  });

  // Round timer: endRound after drawTime seconds
  const timeOut = setTimeout(async () => {
    await endRound(roomId, io, RounEndReason.TIMEUP);
  }, room.settings.drawTime * 1000);
  timers.set(roomId, timeOut);

  // First hint at 50% of drawTime
  if (room.settings.hints > 0) {
    const hintsTimeout = setTimeout(async () => {
      await sendHint(io, roomId);
    }, room.settings.drawTime * 0.5 * 1000);
    hintTimers.set(roomId, hintsTimeout);
  }
}

guessWord() — The Core Guess Logic

This function handles both correct and incorrect guesses:

export async function guessWord(roomId, guess, socket, io) {
  const room = await getRedisRoom(roomId);
  const player = room.players.find(p => p.playerId === socket.id);
  const currentPlayer = room.players[room.gameState.currentPlayer];

  // Correct guess: player is not the drawer, word matches, not already guessed
  if (
    player.playerId !== currentPlayer.playerId &&
    room.gameState.word === guess.toLowerCase() &&
    !player.guessed
  ) {
    player.guessed = true;
    player.guessedAt = new Date();
    await setRedisRoom(room.roomId, room);
    io.to(room.roomId).emit(GameEvent.GUESSED, player);

    // All players guessed (except drawer)? End round early.
    if (room.players.every(p => p.guessed || p.playerId === currentPlayer.playerId)) {
      await endRound(room.roomId, io, RounEndReason.ALL_GUESSED);
    }
  } else {
    // Wrong guess: relay to all players so it shows in chat
    io.to(room.roomId).emit(GameEvent.GUESS, guess, player);
  }
}

Important: The server stores guesses in the guessedWords array but doesn't actually use it anywhere. The chat is managed entirely client-side in MessagesContext.tsx.

endRound() — Transitioning Between Turns

export async function endRound(roomId, io, reason) {
  let room = await getRedisRoom(roomId);
  clearTimers(room.roomId);

  // If reason=LEFT and only 2 players remain, cancel (endGame instead)
  if (reason === RounEndReason.LEFT && room.players.length === 2) return;

  room.gameState.currentPlayer += 1;
  // If all players have drawn, increment round
  if (room.gameState.currentPlayer >= room.players.length) {
    room.gameState.currentRound += 1;
    room.gameState.currentPlayer = 0;
  }

  await givePoints(roomId);  // Calculate and award points
  room = await getRedisRoom(roomId);

  // Reset guessed flags
  room.players = room.players.map(p => ({ ...p, guessed: false, guessedAt: null }));
  room.gameState.drawingData = [];

  // Tell everyone the word and give 5 seconds pause
  io.to(room.roomId).emit(GameEvent.TURN_END, room, { word: room.gameState.word, reason, time: END_ROUND_TIME });

  room.gameState.word = "";
  room.gameState.roomState = RoomState.CHOOSING_WORD;

  setTimeout(async () => {
    if (room.gameState.currentRound > room.settings.rounds) return await endGame(roomId, io);
    await nextRound(roomId, io);
  }, END_ROUND_TIME * 1000);
}

Scoring — givePoints()

export async function givePoints(roomId: string) {
  const room = await getRedisRoom(roomId);
  const now = new Date();
  const playersWhoGuessed = room.players.filter(p => p.guessed);

  // Each guesser gets max(200 - secondsToGuess, 0) points
  playersWhoGuessed.forEach(player => {
    const guessTime = Math.abs(
      (now.getTime() - new Date(player.guessedAt ?? now).getTime()) / 1000
    );
    player.score += Math.round(Math.max(200 - guessTime, 0));
  });

  // Drawer gets 50 base + 10 per guesser
  room.players[room.gameState.currentPlayer].score +=
    DRAWER_POINTS + playersWhoGuessed.length * BONUS_PER_GUESS;
}

Formula: Guessing faster = more points. The drawer always gets at least 50 points per round.

Vote-Kicking

The vote_kickers array is [playerId, voterId[]][]. When a player initiates a vote, the server:

  1. Finds or creates an entry for the target player
  2. Adds the voter's ID (if not already there)
  3. Calculates votesNeeded = Math.ceil(players.length / 2)
  4. If votes ≥ votesNeeded, removes the player from the room and emits KICKED to them

Word Selection

The server reads words from server/words/en.txt (currently only English). The getRandomWords() function:

  1. Loads all words from the file (cached in memory after first load)
  2. If onlyCustomWords is false, blends custom words with file words (custom words get 3× weight)
  3. Fisher-Yates shuffles the array
  4. Returns the first n words

💡 Interview Question: "What happens when a player disconnects mid-game?"

handlePlayerLeft() checks if the leaving player is the current drawer — if so, it ends the round. It cleans up empty rooms. If only 1 player remains mid-game, the game ends. The creator torch passes to the next player in private rooms.
