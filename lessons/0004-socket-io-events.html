Socket.IO Events

The real-time event system: every event name, what triggers it, and what happens server-side.

What You'll Learn

By the end of this lesson, you'll understand the complete event vocabulary between client and server — the 30+ events that power the game.

Event Flow Pattern

Every socket event follows the same pattern:

Client emits event Server handler Server broadcasts ───────────────── ────────────── ───────────────── socket.emit(EVENT, data) → socketHandlers.ts routes → io.to(room).emit(EVENT, data) to roomController fn

The Socket Handler Router

server/src/socket/socketHandlers.ts is the event router. It maps event names to handler functions:

setupSocket(io) {
  io.on("connection", (socket) => {

    socket.on(GameEvent.JOIN_ROOM, (data) => {
      // → handleNewRoom() or handleNewPlayerJoin()
    });

    socket.on(GameEvent.START_GAME, (data) => {
      // → startGame()
    });

    socket.on(GameEvent.DRAW, (data) => {
      // → handleDrawAction("DRAW", data)
    });

    socket.on(GameEvent.DRAW_CLEAR, () => {
      // → handleDrawAction("CLEAR")
    });

    socket.on(GameEvent.DRAW_UNDO, () => {
      // → handleDrawAction("UNDO")
    });

    socket.on(GameEvent.GUESS, (data) => {
      // → guessWord()
    });

    socket.on(GameEvent.WORD_SELECT, (data) => {
      // → wordSelected()
    });

    socket.on(GameEvent.CHANGE_SETTINGS, (data) => {
      // → handleSettingsChange()
    });

    socket.on(GameEvent.DISCONNECT, () => {
      // → handlePlayerLeft()
    });

    socket.on(GameEvent.VOTE_KICK, (data) => {
      // → handleVoteKick()
    });
  });
}

Complete Event Reference

Client → Server Events

EventDataHandlerWhat Happens
JOIN_ROOM {playerData, language, roomId?, isPrivate, password?} handleNewRoom / handleNewPlayerJoin Creates or joins a room. Validates password for private rooms. Adds player to Redis state.
START_GAME {customWords?, onlyCustomWords?} startGame Creator-only. Requires 2+ players. Resets scores, starts round 1.
DRAW {x, y, color, lineWidth, end} handleDrawAction("DRAW") Validates sender is drawer. Pushes stroke to drawingData. Broadcasts to others.
DRAW_CLEAR — handleDrawAction("CLEAR") Clears all drawingData. Broadcasts clear to all.
DRAW_UNDO — handleDrawAction("UNDO") Pops last stroke from drawingData. Broadcasts undo to all.
GUESS {guess: string} guessWord Compares guess to word. Correct → GUESSED + points. Wrong → broadcast as chat.
WORD_SELECT {word: string} wordSelected Sets the word for the turn. Starts draw timer + hint timer.
CHANGE_SETTINGS {setting, value} handleSettingsChange Owner-only. Validates and updates room settings in Redis.
VOTE_KICK {playerId: string} handleVoteKick Records vote. If majority → removes player + emits KICKED.
DISCONNECT — handlePlayerLeft Removes player. Transfers ownership if creator left. Cleans up empty rooms.

Server → Client Events

EventDataSent ToPurpose
JOINED_ROOM {room: Room} Joining player Confirms join. Full room state.
PLAYER_JOINED {player: Player} All in room Broadcasts new player arrival.
PLAYER_LEFT {playerId, room?} All in room Broadcasts player departure. Includes updated room if game ended.
GAME_STARTED {room} All in room Game begins. Resets turn/round display.
CHOOSE_WORD {words: string[], time: number} Drawer only Shows word selection buttons.
CHOOSING_WORD {currentPlayer: Player, time: number} Guessers Shows "Player is choosing..." overlay.
WORD_CHOSEN {word: string, time: number} Drawer only Reveals the actual word to the drawer.
GUESS_WORD_CHOSEN {word: number[], time: number} All Shows underscore lengths. word = [3, 5] for "ice cream".
DRAW_DATA DrawData Guessers Live stroke data. Drawer doesn't receive their own.
CLEAR_DRAW — All Clears canvas.
UNDO_DRAW — All Undoes last stroke.
GUESS {sender, guess} All Wrong guess shown as chat message.
GUESSED {sender, guess} All Correct guess notification.
GUESS_HINT {index, letter} Guessers Reveals one letter in the word.
TURN_END {room, {word, reason, time}} All Turn over. Shows the word.
GAME_ENDED {room} All Game over. Shows final scores.
SETTINGS_CHANGED {setting, value} All Settings updated by owner.
KICKED — Kicked player Forces disconnect.
GAME_STATE {room} Late joiner Mid-game join sync.
Socket.IO Rooms: When a player joins, they're added to a Socket.IO "room" matching the roomId. This lets the server broadcast to all players in a game with io.to(roomId).emit(...) without sending events to other games.
