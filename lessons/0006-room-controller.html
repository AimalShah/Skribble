Room Controller (Game Logic)

The brain of Skribble: every game function in the 662-line file that powers turns, guessing, scoring, and more.

What You'll Learn

By the end of this lesson, you'll understand every function in server/src/game/roomController.ts — the largest and most important file in the server.

Module-Level State

Three Maps track timers that persist across function calls:

timers: Map<string, NodeJS.Timeout>      // Turn timers per room
hintTimers: Map<string, NodeJS.Timeout>  // Hint reveal timers per room
startGameTimers: Map<string, NodeJS.Timeout> // Auto-restart timers for public rooms

Function Reference

Timer Management

FunctionPurpose
clearTimers(roomId) Clears both turn and hint timers. Called before starting new timers.

Game Lifecycle

FunctionPurpose
startGame(room, io) Resets all scores to 0. Sets round=1. Emits GAME_STARTED. Calls nextRound().
endGame(roomId, io) Resets room to NOT_STARTED. Emits GAME_ENDED. For public rooms: auto-restarts after delay.
endRound(roomId, io, reason) Advances to next player. If all players had a turn: advances round. Calls givePoints(). Emits TURN_END with word reveal. Schedules nextRound() or endGame().
nextRound(roomId, io) Gets random words. Sends CHOOSE_WORD to drawer. Sends CHOOSING_WORD to guessers. Starts word-choice timer (auto-selects random word on timeout).

Core Game Actions

FunctionPurpose
guessWord(roomId, guess, socket, io) Serialized. Compares guess to current word (case-insensitive). Correct: marks player.guessed, emits GUESSED, checks if all guessed. Wrong: broadcasts guess as chat.
wordSelected(roomId, word, io) Clears timers. Sets gameState.word. Sends WORD_CHOSEN (full word) to drawer. Sends GUESS_WORD_CHOSEN (underscores) to guessers. Starts draw timer + hint timer.
sendHint(io, roomId) Reveals one random letter. Calculates ceil(word.length/3) positions, picks one. Emits GUESS_HINT. Schedules next hint if more remain.

Scoring

FunctionPurpose
givePoints(roomId) Guesser score: max(200 - secondsToGuess, 0). Drawer score: 50 + (correctGuessers × 10). Updates all player scores.

Player Management

FunctionPurpose
handleNewRoom(io, socket, playerData, language, isPrivate) Non-private: tries to find existing public room with space. Private: creates empty room. Then joins the player.
handleNewPlayerJoin(roomId, socket, io, playerData, language) Serialized. Hydrates from MongoDB if not in Redis. Checks max players. Handles reconnection. Adds player. For public rooms: auto-starts when 2+ join.
handlePlayerLeft(socket, io) Serialized. If drawer left: ends round. Removes player. Transfers ownership if creator left. If only 1 player during game: ends game. If 0 players: deletes room.
handleInBetweenJoin(roomId, socket, io) For late-joining players. Calculates elapsed time. Sends current game state with word as underscores.
handleVoteKick(socket, io, playerId) Serialized. Tracks votes per target. Requires majority (ceil(players/2)). If reached: removes player, emits PLAYER_LEFT + KICKED.

Drawing

FunctionPurpose
handleDrawAction(socket, action, drawData?) Serialized. Validates sender is drawer. DRAW: pushes to drawingData, broadcasts. CLEAR: empties drawingData. UNDO: pops last stroke. Saves to Redis.

Settings

FunctionPurpose
handleSettingsChange(socket, io, setting, value) Serialized. Owner-only. Validates setting name and type. Updates in Redis. Broadcasts SETTINGS_CHANGED.

The Scoring Formula

Guesser points: max(200 - secondsToGuess, 0)
Faster guesses = more points. A guess in 5 seconds = 195 points. A guess in 200+ seconds = 0 points.
Drawer points: 50 + (correctGuessers × 10)
Base 50, plus 10 per player who guessed correctly. With 4 guessers all correct: 50 + 40 = 90 points.
