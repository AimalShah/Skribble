Client Components

Every React component: what it renders, what it listens to, and what it emits.

What You'll Learn

By the end of this lesson, you'll know the role of every component in the client — from pages to game UI to UI primitives.

Component Hierarchy

App.tsx (Router) ├── /login → AuthForm ├── / → HomePage (Navbar + Welcome + Stats) ├── /feed → FeedPage (Navbar + Room Cards) ├── /create-room → CreateRoomPage (Navbar + Form) ├── /profile → ProfilePage (Navbar + Avatar Editor) └── /room/:roomId → GameRoomPage └── RoomProvider └── Game ├── PlayerScores │ └── PlayerCard (× N) │ └── Dialog (mute, kick, invite) ├── GameCanvas (HTML5 Canvas) ├── OverlayContent │ ├── GameSettings (pre-game) │ ├── ChoosingWord │ ├── WordSelector (drawer picks word) │ └── Winners (final scores) ├── Header (timer + word display) ├── GuessInput (mobile) ├── Chat │ └── Message (× N) ├── Toolbar (drawer only) └── AudioManager (invisible)

Pages

PageRouteWhat It Does
HomePage.tsx / Welcome message, quick action cards (Browse/Create), stats dashboard.
FeedPage.tsx /feed Fetches published rooms every 10s. Shows room cards. Public: direct join. Private: password modal.
CreateRoomPage.tsx /create-room Full creation form: name, privacy, password, language, settings, custom words.
GameRoomPage.tsx /room/:roomId Connects socket, emits JOIN_ROOM, renders Game. Posts stats on GAME_ENDED.
ProfilePage.tsx /profile Avatar emoji editor, display name, stats display.

Game Components

ComponentResponsibilitySocket Events
Game.tsx Main layout. Three-column: sidebar | canvas+overlay | chat. Wraps MessagesProvider. —
GameCanvas.tsx HTML5 Canvas with touch+mouse. Drawing, receiving strokes, undo/clear. Auto-resizes. Emits: DRAW, DRAW_CLEAR, DRAW_UNDO. Listens: DRAW_DATA, CLEAR_DRAW, UNDO_DRAW, JOINED_ROOM, WORD_CHOSEN, TURN_END, GAME_ENDED.
Header.tsx Timer (red pulse ≤10s). Word display: text for drawer, underscores+hints for guessers. Listens: WORD_CHOSEN, GUESS_WORD_CHOSEN, CHOOSE_WORD, CHOOSING_WORD, TURN_END, GAME_ENDED, GUESS_HINT, GAME_STATE.
Chat.tsx Chat panel. Sends guesses. Auto-scrolls. Uses MessagesContext. Emits: GUESS. Listens: (via MessagesContext)
Toolbar.tsx Colors (26), line widths (6), undo, clear. Only visible when myTurn + DRAWING. Emits: DRAW, DRAW_CLEAR, DRAW_UNDO.
PlayerScores.tsx Scoreboard. Sorted by score. Shows round info. Renders PlayerCard for each. Listens: PLAYER_JOINED, PLAYER_LEFT, TURN_END.
PlayerCard.tsx Individual card: rank, avatar, name, crown (creator), score. Click → Dialog with mute/kick/invite. Emits: VOTE_KICK.
OverlayContent.tsx State-dependent overlay. NOT_STARTED → settings. CHOOSING_WORD → waiting. PLAYER_CHOOSE_WORD → word buttons. WINNER → scores. —
GuessInput.tsx Mobile-only guess input. Shows character count. Emits: GUESS.
AudioManager.tsx Invisible. Plays sounds: playerJoin.wav, playerLeft.wav, playerGuess.wav. Listens: PLAYER_JOINED, PLAYER_LEFT, GUESSED.
