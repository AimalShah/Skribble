Key Types

TypePropertiesPurpose
Room roomId, creator, players[], gameState, settings, isPrivate, vote_kickers[] Full room state — stored in Redis
Player playerId, name, score, guessed, guessedAt, appearance Extends PlayerData with game runtime state
GameState currentRound, drawingData[], guessedWords[], word, currentPlayer, roomState, timerStartedAt, hintLetters[] Mutable game state within a room
Settings players, drawTime, rounds, customWords, onlyCustomWords, language, wordCount, hints Configurable room settings
DrawData x, y, color, lineWidth, end Single stroke segment sent via socket
GuessedLetters index, letter A single revealed hint character

RoomState Enum

StateMeaning
NOT_STARTEDLobby — waiting to play
PLAYER_CHOOSE_WORDCurrent drawer is picking a word
CHOOSING_WORDOther players see "X is choosing a word"
DRAWINGDrawer is drawing, guessers are guessing
GUESSEDTurn just ended — briefly shows the word
TIMEUPTimer ran out
WINNERGame ended — shows final scores

Scoring Formula

  Guesser score = max(200 - timeToGuessInSeconds, 0)
  Drawer score  = 50 + (numberOfGuessers * 10)

Redis Key Scheme

  Private rooms:  room:<roomId>
  Public rooms:   publicRoom:<roomId>
